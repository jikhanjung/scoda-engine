## 변경 요약

ScodaDesktop v0.1.1 → v0.1.2 버전 업.

### v0.1.2 변경사항

- **P16: Hub 패키지 자동 체크 및 다운로드**
  - `hub_client.py` (순수 stdlib): Hub index fetch, 로컬 비교, SHA-256 검증 다운로드, 의존성 해결
  - GUI "Hub - Available Packages" 섹션: 기동 시 백그라운드 체크, Download / Download All 버튼, 프로그레스 바
  - Hub 섹션이 Log 위에 표시되도록 pack 순서 수정
  - 테스트 25개 추가 (전체 249)
- **GUI 정리**: "Open .scoda File..." 버튼 및 D&D 지원 제거 (Hub으로 대체)
  - `_open_scoda_file()`, `_load_scoda_from_path()`, `_setup_dnd()`, `_on_drop()` 삭제
  - `filedialog`, `scrolledtext` import 제거
  - `--scoda-path` CLI 인자는 유지 (개발/디버깅용)
