# 041: Diff Tree — 단일 병합 트리 with 변경 색상 코딩

**날짜:** 2026-03-07
**Related:** 040 (Side-by-Side sync), trilobase devlog 114

## 개요

두 classification profile 간 차이를 단일 트리에 색상 코딩으로 시각화.

## tree_chart.js 변경

### buildHierarchy()
- `diff_mode` 설정 감지 → `edge_query`로 diff 엣지 로드
- `diffStatusMap`: child_id → {diff_status, parent_id_a, parent_id_b}
- Moved 노드를 `parent_id_b`(compare profile 부모)로 re-parent

### assignColors()
- diff mode: status별 색상 적용 (same=#adb5bd, moved=#fd7e14, added=#198754, removed=#dc3545)

### drawLinks()
- diff 색상 링크: same=faint(0.08 opacity), 나머지=bold colored
- Ghost edge 코드 존재하나 현재 비활성

### drawDiffLegend()
- 우측 하단에 반투명 범례 박스
- Status별 원형 마커 + 카운트

### Tooltip
- `onMouseMove()`, `_showTooltipForNode()` 양쪽에서 diff status 표시
- Moved 노드: `BaseParent → CompareParent` 이동 경로
- `_getNodeLabel(nodeId, labelKey)` 헬퍼 추가
